
# BATTLEDROP – PUBG Esports Platform

Full esports platform starter project focused on PUBG.

Features scaffolded:

- Player profiles
- PUBG account linking
- KD / kill / match stats (API ready)
- Map performance analytics
- AI performance analysis
- Tournament system
- Team system
- Leaderboards
- Esports news system (AI ready)
- Admin dashboard
- Security-ready auth structure
- API services folder
- Auto-update ready structure
- Mobile-ready frontend

Tech Stack (recommended):

Frontend:
Next.js / React

Backend:
Node.js API routes

Auth:
Firebase Auth / Supabase Auth

Database:
PostgreSQL / Firebase / Supabase

PUBG API:
https://developer.pubg.com

AI:
OpenAI API

Run:

npm install
npm run dev
